# 💰 Expense Tracker API

A robust RESTful API built with **Spring Boot** to help users track their daily expenses, manage budgets, and generate financial reports.

---

## 🚀 Features
* **User Authentication:** Secure Sign-up and Login using Spring Security (JWT).
* **Expense Management (CRUD):** Complete control to Create, Read, Update, and Delete expense records.
* **Categorization:** Organize expenses by categories (e.g., Food, Transport, Rent).
* **Reporting:** Generate summaries of spending habits over specific time periods.

---

## 🛠️ Tech Stack

* **Language:** Java 25
* **Framework:** Spring Boot 4.x
* **Database:** MySQL 8.x
* **Build Tool:** Maven
